# 🎬 CineVault – Movie Review & Rating Platform

CineVault is a frontend movie and TV show discovery platform that allows users to explore trending content, search for movies/TV shows, and manage their personal entertainment lists. It integrates with the **TMDB API** to fetch real-time movie data and provides features like rating, reviews, watchlists, and personalized vaults.

## 🚀 Features

- 🔍 Search movies and TV shows using the **TMDB API**
- 📈 View trending and popular content
- ⭐ Rate movies and TV shows
- 📝 Write and manage reviews
- 🎯 Create and manage a personal **Watchlist**
- 📂 Organize saved content into custom **Vaults**
- 💾 Uses **Local Storage** for persistent user data
- 🎨 Responsive and modern UI built with Tailwind CSS

## 📌 Problem Statement

Users often switch between multiple platforms to discover movies, track watchlists, and save reviews. CineVault solves this by providing a single platform where users can discover content and maintain their personal movie records.

---

## ⚙️ How It Works

1. Users search for movies or TV shows.
2. The platform fetches relevant details from TMDB API.
3. Users can:
   - Rate content
   - Add reviews
   - Save to watchlist
   - Organize content into vaults
4. Data is stored locally for easy access even after refreshing the page.
